A visual and interactive web application using NASA’s Asteroid API that shows the user all the asteroids close to Earth on the date selected and providing each asteroids information. It shows asteroids orbiting the Earth with distance accurately scaled 1px : 75000mi.

• Created with: HTML5, CSS3, JavaScript (jQuery, JSON), NASA & WebGL Earth API.
